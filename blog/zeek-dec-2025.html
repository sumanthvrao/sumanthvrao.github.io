Zeek Home Network Scanning

Dec 29, 2025

Zeek (formerly Bro) is a powerful passive network monitoring tool, that provides a lot of tools out of the box for network security monitoring. If you are a network tinkerer like me, and you want to monitor your own home network traffic, Zeek is a great tool to get started with. In this blog, I'll mention how I got this done with pretty cheap ~100$ worth of hardware.

Hardware Configuration

  • - 1x Raspberry Pi 5 Model B (~$50)
  • - 1x ASUS RT-AC66U router (~$100)
  • - 2x ethernet cables (~$10)

The raspberry pi is used to run the Zeek server (and any other tool e.g., pi-hole, etc) you want monitoring your network traffic. The ASUS router acts are a managed switch (software defined) -- mainly by flashing a custom firmware (ASUS Merlin) on it and configuring iptables to forward traffic to the raspberry pi. Understandably, this only works if the volume of traffic is not too high, but for a small home network, this is more than enough.
